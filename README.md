# SiMute
A simple minecraft bukkit mute plugin.

## Using
```/mute <on/off/get> <Player>```


```on```  mute <br/>
```off```  unmute <br/>
```get```  get mute status <br/>
```Player```  player name <br/>

## Compile
Run this in command line:
```
./gradlew build
```
